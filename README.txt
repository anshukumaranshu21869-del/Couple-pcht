CoupleVerse Demo

Files:
- index.html
- styles.css
- script.js

How to upload on GitHub:
1. Open your repository.
2. Tap Add file.
3. Choose Upload files.
4. Upload all three files together, or upload this ZIP after extracting.
5. Commit changes.
6. Open Settings > Pages and deploy from the main branch.

Note:
This is a frontend demo. Real private chat and secure partner accounts will require Firebase Authentication and database security rules.
